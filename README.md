# 💢 ゆりかのぷんぷんゲーム

ツンデレなゆりかちゃんをデレさせる、ちょっと癒し系インタラクティブゲームです。

## ファイル構成

```
yurika-game/
├── index.html        # ゲーム本体
├── yurika-angry.jpg  # ぷんぷん顔（通常・怒り状態）
├── yurika-smile.mp4  # にっこり動画（デレ状態）
└── README.md
```

## 遊び方

`index.html` をブラウザで開くだけ！（ローカルファイルで動作します）

| ボタン | 効果 |
|--------|------|
| 👆 つんつんする | ぷんぷんゲージ大上昇 💢 |
| 😈 からかう | かなり怒らせる（危険） |
| 💐 褒める | デレゲージUP！ |
| 🍰 お菓子あげる | デレ度MAX上昇 |

## エンディング

- 💢 ぷんぷんゲージ100 → ゆりかに絶縁される…
- 🥰 デレゲージ90以上 → 「…すき」ハッピーエンド！

## 注意

動画再生はブラウザのautoplay制限により、**ユーザー操作後**に再生されます。  
ローカルで開く場合はChromeまたはSafariを推奨。
